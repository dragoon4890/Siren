üîó How to Use the TTS API

Endpoint: POST http://localhost:9090/tts

Request Body (JSON):

{
  "text": "Your text here",
  "language": "ja-JP"
}

cURL Example:

curl -X POST http://localhost:9090/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hello world","language":"en-US"}' \
  --output speech.mp3

JavaScript Example:

const response = await fetch('http://localhost:9090/tts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    text: 'Hello world',
    language: 'en-US'
  })
});

const audioBlob = await response.blob();
const audioUrl = URL.createObjectURL(audioBlob);
// Use audioUrl with audio element

Response: MP3 audio file (Content-Type: audio/mpeg)
